Targeted Therapies

Since the age of low cost genetic sequencing is upon us, we can now focus on creating therapies that are designed to target key genetic mutations in diseases like cancer. We can sequence these tumors and get the types of mutations that distinguish them from healthy cells. Then we can create therapies that target these key mutations. We have seen this space evolve over the past decade with breakthroughs like KRAS G12C. There are so many other mutations out there that can be addressed with these therapies like PI3Ka. There are hundreds of proteins and enzymes inside a cell that regulate the process of growth and proliferation. Many of them mutate in different cancers. We can use genetic information and other technologies like tyrosine kinase inhibitors and protein degraders to go after these specific mutations that separate cancer cells from healthy cells. This brings with it a future where we can combine older therapies, with these Personalized Therapies and even other technologies like cell therapies. When it comes to taking on very hard diseases like cancer, we can use all the tools we can get.
