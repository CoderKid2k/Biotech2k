Recursion Pharma

Profile

Recursion is using robots to do experiments and collect data. They are using technology to map biology. They have a lab where they can conduct millions of experiments every week. All that data is collected into their Biohive supercomputer. This can be mined by using Artificial Intelligence and Machine Learning. They can collect all the genotype and phenotype data by using CRISPR to knock out each gene one at a time and recording all the changes. This could lead us to discovering new connections between genetics and biology.

Pipeline

REC-4881 - MEK1/2 Phase 2

REC-2282 - NF2 Phase 2

REC-994 - CCM Phase 2

REC-3964 Cdif Phase 1

Science

REC-4881 - This is a specialized MEK1/2 inhibitor which targets tumor growth in the GI tract. The early phase 1 data was for safety, but there is literature that warrants testing of MEK1/2 in colorectal cancers. We will have to see the data to see how well it works.

REC-2282 - This is a HDAC inhibitor focused on targeting NF2. The early phase 1 data showed safety, but now it has to show efficacy in this disease.

REC-994 - This is a selective Superoxide Dismutase inhibitor. This plays a role in oxidative stress in cells. Their early phase 1 data established safety. Now we need to see the efficacy data.

REC-3964 - This is designed to inhibit the toxin that is released by the C diff bacteria that infects the GI tract during infection. This is just entering phase 1 so there is no data yet.
